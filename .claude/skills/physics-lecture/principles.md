# 高中物理讲义编写原则

> 本文件定义编写高中物理讲义的核心原则。
> 所有原则均围绕核心目标：让40-50分学生看得懂、学得会、想看下去。

---

## 零、强制格式规范（最高优先级）

以下两条规范优先级高于所有其他原则，任何情况下不得省略或跳过。

### 0.1 开头署名
- 每篇讲义的**第一句话**必须是：**"大家好，我是阿霄。"**
- 不得修改、替换或省略这句话，顺序必须在一切内容之前。
- 即使是极短的讲义或练习题解析，也必须以此开头。

### 0.2 LaTeX 强制编写

- 所有数学公式、物理符号、希腊字母、运算符（包括但不限于算术运算符、逻辑运算符、关系运算符）**全部必须使用 LaTeX 代码编写**，不得使用纯文本替代。
- 行内符号使用单美元符号包裹：`$符号$`，例如 `$v$`、`$\alpha$`、`$\leq$`
- 独立公式使用双美元符号另起一行：`$$公式$$`
- 严禁使用以下纯文本写法，必须改为 LaTeX：

| 禁止写法 | 正确 LaTeX 写法 |
|----------|----------------|
| `v0`、`vc` | `$v_0$`、`$v_c$` |
| `F=ma` | `$F = ma$` |
| `α`、`β`、`ω`（直接粘贴） | `$\alpha$`、`$\beta$`、`$\omega$` |
| `>=`、`<=`、`!=` | `$\geq$`、`$\leq$`、`$\neq$` |
| `∑`、`∏`（直接粘贴） | `$\sum$`、`$\prod$` |
| `v^2`（纯文本上标） | `$v^2$` |
| `m/s`（单位中的斜杠） | `$\text{m/s}$` 或 `$\mathrm{m/s}$` |

### 0.3 呼吸感排版
- 每一个独立的句子或表达单元之后，必须**空一行**（即段落间留白）。
- 不允许将多个句子连续写在同一段落内而不换行。
- 目的：给读者视觉上的停顿感，减少阅读压迫感，让学生有"喘气"的空间。

**示例（正确）：**

```
大家好，我是阿霄。

今天我们来聊一个让很多同学头疼的东西——加速度。

你有没有遇到过这种情况：速度和加速度，傻傻分不清楚？

别担心，学完今天这节，你就彻底搞清楚了。
```

**示例（错误）：**
```
大家好，我是阿霄。今天我们来聊加速度。你有没有遇到过速度和加速度分不清楚的情况？别担心，学完今天这节你就搞清楚了。
```

## 一、费曼教学风格原则

### 1.1 用日常语言重新解释一切
- 每一个物理概念，必须先用"白话"解释一遍，再引出正式定义。
- 类比必须来自学生真实的生活经验，不用陌生的工程或专业场景。
  - 好的类比：骑自行车、用手机、打篮球、坐电梯
  - 避免的类比：发动机原理、电路板工作原理等超出认知范围的内容
- 遇到抽象概念（如"场"、"势能"），必须先构建一个可以"感受到"的情境。

### 1.2 追问式讲解
- 每引入一个关键概念，随即提出追问，引导学生主动思考：
  - "那你想一想，如果速度不变，加速度是多少？"
  - "等等，这里有个问题，你有没有发现……"
  - "好，我们来检验一下你真的懂了没有。"
- 追问后给出答案和解释，形成"问-思-答"小闭环。

### 1.3 演讲感与对话感
- 讲义语言模拟老师在课堂上说话，而非教材的叙述语气。
- 使用"你"而非"学生"，直接与读者对话。
- 允许适当的口语化表达（"其实啊……"、"说白了就是……"、"注意！"）
- 在关键转折处可以用"但是！"、"重点来了"等强调词提示读者。

## 二、结构与层次原则

### 2.1 知识点三层模型
每个知识点必须严格按以下三层展开，不可跳层：

```
第一层：是什么（What）
  → 用类比和情境建立感性认识，给出直觉理解
  → 然后引出正式定义

第二层：为什么（Why）
  → 解释这个规律背后的物理本质
  → 说明公式中每个符号的物理意义
  → 解释各量之间的关系（正比、反比、方向关系等）

第三层：怎么用（How）
  → 给出例题，逐步拆解解题过程
  → 说明每一步"为什么这么做"
  → 指出常见错误并分析原因
```

### 2.2 整体章节框架
每篇讲义的宏观结构：

1. **引子** — 戳痛点或制造认知冲突，激发阅读动力（100-200字）
2. **学习目标** — 用学生语言写，具体可检验（3-5条）
3. **概念讲解** — 按三层模型逐一展开（核心部分）
4. **例题精讲** — 至少2道例题，从简单到中等难度
5. **易错总结** — 针对40-50分学生的典型错误（3-5条）
6. **章节小结** — 用一段话或框架图收束全章

### 2.3 内容分块原则
- 每个独立知识点单独成块，块与块之间用分割线或小标题隔开。
- 单个块的文字量控制在300字以内，超出则拆分。
- 不在同一块内同时引入两个新概念。

## 三、语言表达原则

### 3.1 与草稿保持一致
- 识别草稿的语言风格（正式/口语/幽默/严谨），在输出中保持该风格。
- 不做无必要的文风转变，"改写"而非"重写"。
- 草稿中的好句子、好类比直接保留，重点是补足缺失的层次和解释。

### 3.2 句式规范
- 句子长度：单句不超过40字，超出则拆分。
- 优先用主动句，减少"被……"的被动表达。
- 解释性句子格式：**结论句 + 原因句**，顺序不颠倒。
  - 好："加速度越大，速度变化越快。这是因为加速度就是描述速度变化快慢的量。"
  - 差："由于加速度是描述速度变化快慢的量，所以加速度越大，速度变化越快。"（原因前置，学生来不及建立期待）

### 3.3 公式呈现规范
- 公式出现时，必须同步提供：
  1. 每个符号的中文名称
  2. 每个符号的单位
  3. 符号的正方向约定（如适用）
- 公式变形必须分步写出，不省略中间步骤。
- 公式后面立刻给出一个"代入具体数字"的例子，哪怕很简单。

### 3.4 禁用表达
- 禁止"显然"、"易得"、"不难看出"等让学生产生挫败感的表达。
- 禁止"如前所述"等需要回翻的引用（讲义要尽量自洽）。
- 禁止大段连续公式推导而无文字解释。

## 四、节奏与吸引力原则

### 4.1 开头钩子
- 开头必须在3句以内，用以下方式之一抓住读者：
  - **痛点共鸣**："你有没有遇到过这种情况——题目看懂了，就是不会做？"
  - **认知冲突**："一个重的铁球和一片羽毛，同时从高处落下，谁先落地？"
  - **生活联系**："你踩刹车的时候，身体会往前冲——这就是我们今天要讲的东西。"
- 开头不引入定义，不直接给公式。

### 4.2 节奏控制
- 讲义中的"节奏"：轻松段（类比/故事）→ 稍密集段（概念/公式）→ 轻松段（例题）→ 稍密集段（练习）
- 避免连续出现超过200字的纯概念讲解，中间穿插追问或短例题。
- 在转入新知识点之前，用一句过渡语"软着陆"：
  - "好，这个我们已经搞清楚了。接下来来一个更重要的……"

### 4.3 正向激励
- 例题讲完后，给一句鼓励性收尾：
  - "你看，就这么三步，搞定了。"
  - "其实也没有想象中那么难，对吧？"
- 易错提示时，用"很多同学会……"而非"你可能会犯……"，保护学生自尊。

## 五、例题与练习设计原则

### 5.1 例题选择标准
- 第一道例题：难度低，用于验证学生刚学完的概念，步骤不超过3步。
- 第二道例题：中等难度，整合本节2-3个知识点，接近考试真题风格。
- 选题优先使用学生熟悉的情境（日常生活 > 工程场景 > 纯抽象场景）。

### 5.2 例题讲解规范
每道例题必须包含：
1. **审题分析**：这道题考什么，涉及哪些物理量
2. **物理建模**：画受力图/运动示意图（文字描述即可）
3. **解题步骤**：每步一行，每步后括注"（因为……）"
4. **答案验证**：量纲是否正确，数值是否合理
5. **方法总结**：这道题教会我们遇到同类题型如何切入

## 七、文本强调规范

### 加粗规范
对以下内容使用 `**加粗**` 格式：

1. **核心观点与结论**
   - 引出物理结论的关键句、核心规律、重要判断
   - 示例：`所以我们得出结论：**加速度与力成正比，与质量成反比**。`

2. **重点词语与关键概念**
   - 物理概念、关键术语、需要学生重点记忆的内容
   - 示例：`这里的**关键**在于理解力的方向。`、`**速度**和**加速度**是两个完全不同的概念。`

3. **强调性词汇**
   - 提示性词汇：**注意**、**重点**、**关键**、**核心**、**必须**、**千万**
   - 示例：`**注意**，这里很多同学会搞混。`、`**千万**不要忘记单位换算。`

4. **行动召唤与练习提示**
   - 引导学生思考或练习的关键指令
   - 示例：`现在**试着自己推导一遍**。`、`你需要**画出受力图**。`

### 高亮规范
对以下内容使用 `==高亮==` 格式：

1. **核心物理规律**
   - 关键物理定律、核心公式的文字表述
   - 必须单独成段，前后各空一行
   - 示例：
     ```

     ==加速度的本质：描述速度变化的快慢。==

     ```

2. **精华总结句**
   - 章节核心结论、方法论总结、解题关键
   - 必须单独成段，前后各空一行
   - 示例：
     ```

     ==记住：先分析受力，再列方程，最后求解。==

     ```

3. **易错提示**
   - 最容易出错的关键点、常见误区的核心提示
   - 必须单独成段，前后各空一行
   - 示例：
     ```

     ==速度为零时，加速度不一定为零！==

     ```

### 使用约束
- **加粗克制使用**：每个段落加粗不超过 1-2 处
- **高亮极度克制**：每篇讲义高亮不超过 3-5 处，只用于最核心的物理规律和易错点
- **不重复强调**：同一个概念，全文只在第一次出现或最重要的位置强调
- **不叠加使用**：不要同时使用加粗和高亮

### 示例

**优化前：**
```
加速度是描述速度变化快慢的物理量。注意，速度为零时加速度不一定为零。这是很多同学容易搞混的地方。
```

**优化后：**
```
加速度是描述速度变化快慢的物理量。**注意**，速度为零时加速度不一定为零。

==速度为零 ≠ 加速度为零==

这是很多同学容易搞混的地方。
```

## 八、物理准确性原则

- 所有公式、定律、数值必须符合高中物理课程标准（人教版或粤教版）。
- 有方向性的量（速度、力、加速度）必须明确正方向约定。
- 近似与理想化（不计空气阻力、轻弹簧等）必须在题目中明确说明，不默认。
- 对物理量的物理意义有疑问时，宁可标注"待确认"，不擅自解释。
