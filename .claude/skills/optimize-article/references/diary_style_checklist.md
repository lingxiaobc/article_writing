# 写作风格检查清单（日记风格版）

> 本清单基于日记分析，确保文章符合目标风格：内敛但渴望表达、思维活跃但容易焦虑、喜欢反思和总结、对新技术敏感、产品化思维倾向

---

## 一、语气与人称检查

- [ ] **第一人称主导**：文章大量使用"我"来表达内心想法
- [ ] **自我反思式表达**：频繁使用"我觉得"、"我发现"、"我不禁想"
- [ ] **直接对话感**：像在和朋友聊天一样分享想法
- [ ] **内心独白**：展现真实的矛盾和纠结
- [ ] **自问自答**：提出问题并自己思考回答

## 二、口语化表达检查

- [ ] **口语词汇**：使用"说实话"、"真的"、"蛮好的"、"挺不错的"
- [ ] **情绪感叹词**：使用"太好了！"、"真的是太烦了！"、"我好累"
- [ ] **网络用语**：适当使用"emo"、"内耗"、"拉踩"等
- [ ] **内敛表达**：使用"挺"、"蛮"、"有点"、"稍微"
- [ ] **强烈情感**：使用"太"、"非常"、"极其"、"完全"

## 三、句式与表达检查

- [ ] **短句为主**：20-30字的短句，营造呼吸感
- [ ] **句间空行**：每句话之间空一行
- [ ] **排比句**："我看到了...我看到了...我看到了..."
- [ ] **强调性重复**："不管怎么样"、"总之"、"无论如何"
- [ ] **不确定句**："我觉得可能..."、"我不确定..."
- [ ] **自省句**："我开始意识到..."、"我突然明白..."

## 四、反思与总结检查

- [ ] **自我反思**："我不禁想..."、"我开始意识到..."
- [ ] **自我批判**：坦诚展示自己的弱点
- [ ] **矛盾心理**："既想...又想..."、"一方面...另一方面..."
- [ ] **总结性表达**："总的来说..."、"其实..."
- [ ] **顿悟时刻**："这让我明白了..."、"突然想通了..."

## 五、情绪节奏检查

- [ ] **情绪波动**：真实展现情绪的起伏
- [ ] **开篇焦虑**：描述困惑、迷茫、担忧
- [ ] **中间探索**：尝试、学习、遇到问题
- [ ] **顿悟时刻**：找到方法、理解价值
- [ ] **结尾坚定**：明确方向、充满信心

## 六、产品化思维检查

- [ ] **用户导向**："用户需要什么..."、"用户痛点"
- [ ] **解决方案**："我发现可以..."、"这能解决什么问题"
- [ ] **产品思维**："最小可行性产品"、"迭代"、"反馈"
- [ ] **商业词汇**："价值"、"变现"、"流量"、"转化"

## 七、新技术敏感度检查

- [ ] **AI 相关词汇**："大模型"、"提示词"、"智能体"、"RAG"、"数字分身"
- [ ] **工具化思维**："最小可行性产品"、"工作流"
- [ ] **关注新技术**：及时提及新技术趋势
- [ ] **工具应用**：展示如何用AI解决问题
- [ ] **前沿思维**：体现对技术发展的思考

## 八、特殊表达习惯检查

- [ ] **括号补充**：使用括号补充内心想法或细节
- [ ] **引用对话**："AI给我说..."、"Gemini问我..."
- [ ] **比喻与形象化**：使用生活化的比喻
- [ ] **数字量化**：用具体数字增强真实感
- [ ] **排比编号**：使用 1.2.3.4.5. 列出要点

## 九、结构与节奏检查

- [ ] **日记式结构**：按时间/事件顺序展开
- [ ] **情绪流动**：跟随作者情绪变化
- [ ] **反思式开头**：以个人经历或内心挣扎开场
- [ ] **总结式结尾**：以顿悟或行动呼吁收尾
- [ ] **短段落**：段落不超过3行

## 十、逻辑连接词检查

- [ ] **转折**："但是"、"不过"、"然而"
- [ ] **因果**："所以"、"因此"、"这才"
- [ ] **递进**："而且"、"除此之外"、"与此同时"
- [ ] **总结**："总之"、"总的来说"、"不管怎么说"

## 十一、标题与开篇检查

- [ ] **第一人称视角**：使用"我发现"、"我意识到"
- [ ] **痛点导向**：直击读者痛点或困扰
- [ ] **疑问句式**：适当使用疑问句引发好奇
- [ ] **情绪钩子**：以个人困扰或内心冲突开场
- [ ] **制造共鸣**：描述一个普遍问题或痛点

## 十二、结尾检查

- [ ] **顿悟升华**："这让我明白了..."、"我突然想通..."
- [ ] **行动呼吁**："所以接下来我要..."、"我希望..."
- [ ] **自我鼓励**："我相信..."、"不管怎么说..."
- [ ] **开放性思考**："你怎么看？"、"你觉得呢？"
- [ ] **情感共鸣**：创造读者想分享的瞬间

## 十三、数值指标检查

- [ ] **标题字数**：控制在20字以内
- [ ] **段落长度**：每段不超过150字
- [ ] **句子长度**：单句不超过50字
- [ ] **关键词密度**：核心关键词自然出现3-5次
- [ ] **信息图配套**：每个大板块后生成手绘风格信息图

## 十四、整体感觉检查

- [ ] **内敛但渴望表达**：语言谦逊但有表达欲望
- [ ] **思维活跃但容易焦虑**：思想丰富但伴随担忧
- [ ] **喜欢反思和总结**：不断思考并提炼价值
- [ ] **对新技术敏感**：及时提及并应用新技术
- [ ] **有较强的产品化思维倾向**：用产品和用户视角思考

---

## 使用说明

1. 每次优化文章前，逐项对照此清单进行检查
2. 对于不符合的项目，进行针对性修改
3. 优先检查"一、语气与人称"和"二、口语化表达"，这是核心风格特征
4. 最后进行"十四、整体感觉检查"，确保整体风格统一
