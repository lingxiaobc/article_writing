# 文章质量检查清单

> 执行优化前，使用此清单对原文进行逐项评估。每项标注 ✅ 通过 / ❌ 需改进 / ➖ 不适用。

---

## 一、标题
- [ ] 标题准确概括文章核心内容
- [ ] 标题具有吸引力或引发好奇
- [ ] 标题字数合理（中文建议10-25字）
- [ ] 无标题党或夸大表述

## 二、开头
- [ ] 开篇在3句内提出核心问题或观点
- [ ] 有明确的读者痛点或场景切入
- [ ] 开头不冗长，直接进入主题

## 三、结构与逻辑
- [ ] 全文有清晰的论述主线
- [ ] 各章节之间有自然的逻辑过渡
- [ ] 论点有对应的论据支撑
- [ ] 无重复或矛盾的内容段落
- [ ] 层级标题使用合理（H2/H3层次清晰）

## 四、语言表达
- [ ] 句子简洁，无过度堆砌的修饰词
- [ ] 关键术语使用一致，无前后矛盾
- [ ] 无歧义表达或模糊措辞
- [ ] 语气统一（正式/非正式贯穿全文）
- [ ] 无明显语法错误或错别字

## 五、段落
- [ ] 每段只表达一个核心观点
- [ ] 段落长度适中（中文建议100-200字）
- [ ] 段落首句为该段中心句

## 六、结尾
- [ ] 结尾有明确的总结或收束
- [ ] 有引导读者行动的内容（CTA）（如适用）
- [ ] 结尾不突兀，与全文基调一致

## 七、可信度
- [ ] 数据、引用有来源（如适用）
- [ ] 无无法验证的绝对化表述
- [ ] 专有名词拼写/写法正确

## 八、格式与排版
- [ ] 使用了列表、加粗等适当的排版手段
- [ ] 图片/表格有标注（如适用）
- [ ] 适合目标平台的阅读习惯（PC/移动端）
